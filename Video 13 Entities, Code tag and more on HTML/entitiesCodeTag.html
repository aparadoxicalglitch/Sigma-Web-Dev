HTML Entities, Code Tag, and More

1. HTML Entities

HTML entities are used to display characters that have a special meaning in HTML, or that are not on a standard keyboard.

  • &lt; displays as: <
  • &gt; displays as: >
  • &amp; displays as: &
  • &quot; displays as: "
  • &copy; displays as: ©
  • &euro; displays as: €

3. The <pre> Tag with <code>

For displaying multiple lines of code with indentation, use <pre> and <code> together:

<html>
  <head>
    <title>Sample Page</title>
  </head>
  <body>
    <p>Hello, world!</p>
  </body>
</html>

4. Quoting Text

Use <q> for short inline quotes like this: This is a short quote.

Use <blockquote> for longer or more formal quotations:

The journey of a thousand miles begins with one step.
